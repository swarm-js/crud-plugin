## About The Project

This package aims to help developers quickly create CRUD controllers.

## Built With

- TypeScript@4

## Getting Started

### Installation

```sh
yarn add @swarmjs/crud
```

or

```sh
npm install --save @swarmjs/crud
```

## Usage

```ts
import { Crud } from '@swarmjs/crud'
```
